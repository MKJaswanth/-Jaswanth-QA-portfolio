# Jaswanth QA Portfolio

A bespoke static portfolio website for Jaswanth M, focused on QA, manual testing, defect reporting, and Playwright automation expertise.

## Files

- `index.html` - portfolio content and structure
- `styles.css` - custom visual design and responsive layout
- `script.js` - case-study filtering interaction

Open `index.html` directly in a browser or publish the repository with GitHub Pages.
